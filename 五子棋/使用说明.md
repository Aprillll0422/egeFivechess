# 五子棋游戏 使用说明

## 一、环境要求

本程序基于 **EasyX 图形库** 开发，运行前请确保已安装以下环境：

- **操作系统**：Windows
- **编译器**：Visual Studio 或 MinGW（支持 C++）
- **图形库**：[EasyX 图形库](https://easyx.cn/)（请下载并安装对应编译器版本）

## 二、安装 EasyX 图形库

1. 访问 EasyX 官网：https://easyx.cn/
2. 下载对应你编译器版本的 EasyX 安装包
   - Visual Studio 用户：下载 `EasyX for Visual Studio`
   - MinGW 用户：下载 `EasyX for MinGW`
3. 运行安装程序，按照提示完成安装

## 三、文件结构

运行程序前，请确保以下文件位于**同一目录**下：

```
五子棋/
├── 五子棋.exe      # 可执行程序
├── 五子棋.cpp      # 源代码（如需编译）
├── 黑子1.png       # 黑棋图片
├── 白子1.png       # 白棋图片
├── button.png      # 按钮图片
├── lock.png        # 悔棋标记图片
├── 弹框.png        # 结算弹框图片
├── 弹框小.png      # 结算弹框内部图片
├── bgm.mp3         # 背景音乐
├── 落子.mp3        # 落子音效
├── win.mp3         # 胜利音效
└── equal.mp3       # 平局音效
```

**重要**：所有资源文件（图片、音乐）必须与可执行程序放在同一目录下，否则程序无法正常加载资源。

## 四、运行程序

### 方式一：直接运行
双击 `五子棋.exe` 即可开始游戏。

### 方式二：自行编译
1. 打开 Visual Studio 或其他 IDE
2. 创建新的控制台项目
3. 将 `五子棋.cpp` 添加到项目
4. 确保已安装 EasyX 图形库
5. 编译并运行

## 五、游戏操作

### 游戏模式
程序启动后，可选择以下模式：
- **双人对战**：两名玩家轮流下棋
- **人机对战**：玩家执黑先行，AI 执白后手
- **退出**：退出程序

### 操作说明
- **落子**：鼠标左键点击棋盘空位
- **悔棋**：点击"悔棋"按钮
  - 双人模式：撤销上一步
  - 人机模式：撤销玩家和 AI 各一步，回到玩家回合
- **退出**：点击"退出"按钮

### 游戏规则
- 黑棋先行
- 五子连珠（横、竖、斜）即获胜
- 棋盘填满则平局

## 六、常见问题

### Q1: 程序启动后黑屏或闪退
- 检查所有资源文件是否与 exe 在同一目录
- 确认图片和音乐文件名是否正确（区分大小写）

### Q2: 编译错误，找不到 graphics.h
- 说明未安装 EasyX 图形库，请按第二步安装

### Q3: 没有声音
- 检查 mp3 文件是否存在
- 确认系统音量未被静音

## 七、技术信息

- 开发语言：C++
- 图形库：EasyX
- AI 算法：基于棋型评估的启发式搜索

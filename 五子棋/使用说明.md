# 五子棋游戏 使用说明

## 一、环境要求

本程序基于 **EGE 图形库** 开发，运行前请确保已安装以下环境：

- **操作系统**：Windows
- **IDE**：Dev-C++（推荐版本 5.11）
- **图形库**：EGE 图形库

## 二、安装 Dev-C++ 和 EGE 图形库

### 步骤 1：下载 Dev-C++

1. 访问 Dev-C++ 下载页面（推荐使用 Embarcadero Dev-C++ 或 Orwell Dev-C++ 5.11）
2. 下载安装包并运行
3. 按照安装向导完成安装（建议安装到默认路径）

### 步骤 2：下载 EGE 图形库

1. 访问 EGE 官网：https://xege.org/
2. 点击"下载"页面，下载最新版本的 EGE 图形库
   - 选择 `ege19.xx_all_win.zip`（适用于 Windows，包含 Dev-C++ 所需文件）
3. 解压下载的压缩包

### 步骤 3：配置 EGE 到 Dev-C++

1. 找到 Dev-C++ 的安装目录（默认为 `C:\Program Files (x86)\Dev-Cpp`）
2. 将 EGE 解压后的文件复制到对应目录：
   - **头文件**：将 `include` 文件夹中的所有文件复制到 `Dev-Cpp\MinGW64\include\` 目录
   - **库文件**：将 `lib` 文件夹中的所有 `.a` 文件复制到 `Dev-Cpp\MinGW64\lib\` 目录
3. 或者在 EGE 解压包中运行 `install_for_devcpp.bat` 自动安装脚本（如有提供）

### 步骤 4：Dev-C++ 项目配置

每次创建新项目时需要进行以下配置：

1. 打开 Dev-C++，创建新项目：`文件` → `新建` → `项目` → `Empty Project` → `C++ 项目`
2. 打开项目选项：`项目` → `项目属性`（或按 `Alt + P`）
3. 切换到 `参数` 选项卡
4. 在 `链接器` 栏中添加：
   ```
   -lgraphics64
   -luuid
   -lmsimg32
   -lgdi32
   -limm32
   -lole32
   -loleaut32
   -lwinmm
   -lgdiplus
   ```
5. 点击"确定"保存设置

### 步骤 5：验证安装

创建一个测试程序验证 EGE 是否安装成功：

```cpp
#include <graphics.h>

int main()
{
    initgraph(640, 480);
    outtextxy(100, 100, "EGE 安装成功！");
    getch();
    closegraph();
    return 0;
}
```

如果编译运行成功并显示窗口，说明 EGE 已正确安装。

## 三、文件结构

运行程序前，请确保以下文件位于**同一目录**下：

```
五子棋/
├── 五子棋.exe      # 可执行程序
├── 五子棋.cpp      # 源代码（如需编译）
├── 黑子1.png       # 黑棋图片
├── 白子1.png       # 白棋图片
├── button.png      # 按钮图片
├── lock.png        # 悔棋标记图片
├── 弹框.png        # 结算弹框图片
├── 弹框小.png      # 结算弹框内部图片
├── bgm.mp3         # 背景音乐
├── 落子.mp3        # 落子音效
├── win.mp3         # 胜利音效
└── equal.mp3       # 平局音效
```

**重要**：所有资源文件（图片、音乐）必须与可执行程序放在同一目录下，否则程序无法正常加载资源。

## 四、运行程序

### 方式一：直接运行
双击 `五子棋.exe` 即可开始游戏。

### 方式二：自行编译
1. 打开 Dev-C++
2. 创建新项目：`文件` → `新建` → `项目` → `Empty Project` → `C++ 项目`
3. 将 `五子棋.cpp` 添加到项目
4. 按照第二部分的步骤 4 配置链接器参数
5. 编译并运行（按 `F11` 编译运行）

## 五、游戏操作

### 游戏模式
程序启动后，可选择以下模式：
- **双人对战**：两名玩家轮流下棋
- **人机对战**：玩家执黑先行，AI 执白后手
- **退出**：退出程序

### 操作说明
- **落子**：鼠标左键点击棋盘空位
- **悔棋**：点击"悔棋"按钮
  - 双人模式：撤销上一步
  - 人机模式：撤销玩家和 AI 各一步，回到玩家回合
- **退出**：点击"退出"按钮

### 游戏规则
- 黑棋先行
- 五子连珠（横、竖、斜）即获胜
- 棋盘填满则平局

## 六、常见问题

### Q1: 程序启动后黑屏或闪退
- 检查所有资源文件是否与 exe 在同一目录
- 确认图片和音乐文件名是否正确（区分大小写）

### Q2: 编译错误，找不到 graphics.h
- 说明未安装 EGE 图形库，请按第二步安装

### Q3: 没有声音
- 检查 mp3 文件是否存在
- 确认系统音量未被静音
